# CLAUDE.md

This directory contains brand icons for Snowbridge

## Icons
- `favicon.svg` - Default browser favicon (SVG format)
- `favicon.png` - Fallback browser favicon (PNG format)
- `icon-light.svg` - Icon for light themes
- `icon-dark.svg` - Icon for dark themes
- `icon-dark-master.inkscape.svg` - Master copy of icon for dark themes (Inkscape SVG format)
- `icon-light-master.inkscape.svg` - Master copy of icon for light themes (Inkscape SVG format)

Note that the icon at `/public/images/snowbridge-icon-light.svg` is a copy of `icon-light.svg`

## Commands

* Convert SVG icon to a 16x16 PNG
```
inkscape <SOURCE> --export-width=16 --export-filename=<DEST>
```
